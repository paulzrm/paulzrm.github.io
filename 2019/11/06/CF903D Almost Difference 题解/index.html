CF903D Almost Difference 题解

发表于 2019-11-06 分类于 题解

本文迁移自洛谷原文。

这是什么题目啊，竟然回爆long long…..

这里介绍一个不用压位的方法

思路：

分类讨论即可

1.直接加上所有的y-x

2.单独处理所有的|x-y|$\le$1的情况

复杂度:O(n)

map是O(n log n)

注意:对于爆long long 的情况，可以用long double

只不过会慢一点，而且精度只比long long 大一点
