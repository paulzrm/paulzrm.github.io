高精度对于 Python 来说不是个事。

思路与原版相同。

```python
import math
n = int(input())
l = int(math.log(n - 1, 2)) + 1
res = n * l
res = res - 2 ** l
res = res + 1
t = n * (n - 1) // 2
print(min(res, t) % 100000007)
```
