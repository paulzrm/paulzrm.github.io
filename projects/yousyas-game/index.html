Yousya's Game

《勇者的游戏》是一款 Windows 平台的 C++ 控制台剧情 RPG，由我在 2026 年 1 月至 2 月独立完成。

GitHub Repository 剧情设定（剧透）

项目概述

玩家扮演一名被召唤到异世界的“勇者”，在彩色字符地图中移动、收集金币、躲避或对抗敌人，并随着任务推进逐渐接近故事的真相。

游戏的灵感来自《安德的游戏》式的叙事错位：玩家所理解的世界，与程序背后真正记录的世界并不完全相同。项目还设计了剧情分支和多个结局，让玩家此前的行动影响最后的选择。

玩法与系统

World彩色字符地图

使用 ANSI 颜色和实时控制台刷新表现地图、玩家、武器、敌人和资源。

Combat方向性战斗

移动和武器朝向相互独立，碰撞关系决定攻击与受伤。

Enemy AI视野与追踪

敌人具有方向、视野范围、移动频率和特殊生成逻辑。

Progress存档与多结局

支持新游戏、继续、读取存档，以及已解锁结局查看。

操作

  • W / A / S / D：移动
  • 方向键：改变武器方向并攻击
  • C：在身后放置障碍
  • F：传送

实现

项目按玩家、敌人、地图、战斗、剧情步骤、结局与文件存档拆分模块。敌人的移动和碰撞判定与固定刷新时钟协同工作，地图则维护角色身体、武器、障碍和金币等不同单元。

原始工程使用本地中文代码页。整理公开版本时，全部源码和文档已转换为 UTF-8；程序启动时会显式设置 Windows 控制台 UTF-8 输入输出，并启用 ANSI 颜色。CMake 配置也会为 MSVC 和 MinGW 分别设置 UTF-8 编译参数。

这是一个完整但仍保留实验感的个人作品：从战斗规则、敌人行为到剧情反转和结局收集，均由我独立设计与实现。
