Yousya's Game

《勇者的游戏》是一款 Windows 平台的 C++ 控制台剧情 RPG，由我在 2026 年 1 月至 2 月独立完成。

Play Web Edition GitHub Repository 剧情设定（剧透）

项目概述

玩家扮演一名被召唤到艾瑞斯世界的“勇者”，在彩色字符地图中移动、收集金币、躲避或对抗魔物，并前往四块古陆寻找传说中的圣器。

游戏包含教程、探索、战斗、任务推进、剧情分支和多结局系统。建议先体验游戏，再阅读单独整理的剧透剧情页。

玩法与系统

World彩色字符地图

使用 ANSI 颜色和实时控制台刷新表现地图、玩家、武器、敌人和资源。

Combat方向性战斗

移动和武器朝向相互独立，碰撞关系决定攻击与受伤。

Enemy AI视野与追踪

敌人具有方向、视野范围、移动频率和特殊生成逻辑。

Progress存档与多结局

支持新游戏、继续、读取存档，以及已解锁结局查看。

操作

  • W / A / S / D：移动
  • 方向键：改变武器方向并攻击
  • C：在身后放置障碍
  • F：传送
这是一个完整但仍保留实验感的个人作品：从战斗规则、敌人行为到剧情反转和结局收集，均由我独立设计与实现。
